# Product Overview

Cuban Chain Customizer is a 3D web application for visualizing and customizing Cuban link chains. Users can configure chain length (7-9 links), adjust materials, modify individual link surfaces, and control spacing between links. The app provides real-time 3D rendering with features like auto-rotation, image capture, and 5-second video recording. It supports both desktop and mobile interfaces with responsive controls.

## Key Features

- Interactive 3D chain visualization with orbit controls
- Chain length customization (7-9 links with dynamic cloning)
- Material selection per link and surface
- Adjustable chain spacing and link offsets
- Configuration save/load (JSON format)
- Image capture with white background
- 5-second video recording (WebM format)
- Mobile-optimized interface with bottom navigation
- Debug mode with camera/model position tracking
